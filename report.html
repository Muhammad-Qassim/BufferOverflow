Introduction

This guide will help you understand and demonstrate a buffer overflow attack. Please note that buffer overflow attacks are illegal and unethical. This guide is for educational purposes only.

Prerequisites

  1. Virtual Machine Setup:

    Download and import the provided virtual machine (VM) to VMware or VirtualBox. The default user account is root with no password.

  2. Program and Files:

    You can download the getscore.c, score.txt, and Redhat9 VM from this link.

  3. Compile and Set Permissions:

    Copy the getscore.c program to the VM or download it from here. 

    gcc getscore.c -o getscore
    chmod 755 getscore
    chmod u+s getscore
    chmod 600 score.txt
  4. Check Permissions: 
    ls -l
  5. Run the Program:

    Execute the program: ./getscore "Mary Doe" 123-45-6789.

    Expected output: "Your score is A+".
